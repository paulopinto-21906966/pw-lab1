HTML5 e CSS

O HTML5 introduz vários novos elementos, que permitem definir a estrutura da página de forma semântica, aqui estão alguns deles

Elemento Foi usado? Onde?
header check Em todas as páginas exceto index.html
hx dangerous Não usei
iframe check Na página multimédia
table check Na página de informação e na html5-css
svg dangerous nas paginas do Lab 4 e posterior
br check Em todas as páginas
figcaption check Usei em todas as página, com a imagem do castelo

CSS

CSS serve para adicionar estilo à nossa página, é aconselhável fazê-lo fora dos ficheiros .html

Seletor Foi usado? Como funciona?
Universal check Estiliza um elemento indepentemente das suas caracteristicas
Por ID check Estiliza o elemento com um certo id
De Classe check Estiliza todos os elementos pertencentes àquela classe
Pseudo-Class check Estado do elemento, por exemplo hover, active ou visited.

Combinadores de Seletores CSS

Os Combinadores de Seletores CSS focam-se na interação/relação entre dois ou mais elementos.

Seletor Foi usado? Como funciona?
Descendentes check Elemento dentro de outro elemento, independentemente do nível
Filhos check Elemento dentro de outro elemento, no nível imediatamente seguinte
Adjacentes dangerous Elemento após outro elemento
Imediatamente Adjacentes dangerous Elemento imediatamente após outro elemento
Agrupados check Estilizar múltiplos elementos da mesma forma
